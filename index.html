HonMonit — Real-time network device monitoring system.

This NOC dashboard allows you to monitor remote devices, view resource usage, manage processes, and send remote commands.

  • Use the search bar to filter devices
  • Toggle Admin Mode for advanced controls
  • Use the Filter button to sort by status
